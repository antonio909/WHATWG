<h1>1.2 Isso é HTML5?</h1>

<i>Esta seção é informativa</i>

<p>Em resumo: Sim.</p>

<p align="justify">
  Em mais detalhes: o termo "HTML5" é amplamente utilizado como um termo da moda (<i>buzzword</i>) para se referir ás tecnologias web modernas, muitas das quais (embora de forma alguma todas) são desenvolvidas no WHATWG.
  Este documento é uma delas; outros estão disponíveis na visão geral de padrões (Standard overview) do WHATWG.
</p>
