<h1>1.5 Escopo</h1>

<i>Esta seção é informativa.</i>

<p align="justify">
  Esta especificação limita-se a fornecer uma linguagem de marcação de nível semântico e APIs de script de nível semântico associadas para a criação de páginas acessíveis na web, abrangendo desde documentos estáticos até aplicações dinâmicas.
</p>

<p align="justify">
  O escopo desta especificação não inclui o fornecimento de mecanismos para a customização da apresentação específica de mídia (embora regras de renderização padrão para navegadores web estejam incluídas ao final desta especificação, e diversos mecanismos para integração com o CSS sejam fornecidos como parte da linguagem).
</p>

<p align="justify">
  O escopo desta especificação não é descrever um sistema operacional completo.
  Em particular, softwares de configuração de hardware, ferramentas de manipulação de imagem e aplicativos que se espera que os usuários utilizem diariamente em estações de trabalho de alto desempenho estão fora de escopo.
  Em termos de aplicativos, esta especificação é direcionada especificamente para aqueles que seriam utilizados pelos usuários ocasionalmente, ou regularmente mas de locais distintos, com baixos requisitos de CPU.
  Exemplos de tais aplicativos inclui sistemas de compras online, sistemas de busca, jogos (especialmente jogos online multiplayer), listas telefônicas ou agendas de endereços públicos, softwares de comunicação (clientes de e-mail, clientes de mensagens instantâneas, softwares de discussão), softwares de edição de documentos, etc.
</p>
